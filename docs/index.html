Bootcamp Helper

Extensão de exemplo (Manifest V3) — conta e destaca links na aba ativa.

Instalação (modo desenvolvedor)

  1. Clone o repositório.
  2. Abra `chrome://extensions` no Chrome.
  3. Ative Developer mode e clique em Load unpacked.
  4. Selecione a pasta do projeto (a raiz onde está o `manifest.json`).
Download (.zip) — (adicione sua Release com arquivo zip no GitHub Releases).

Recursos

  • Popup interativo com contagem e destaque
  • Service Worker para eventos e armazenamento
  • Content script para páginas específicas
  • Permissões: storage, activeTab, scripting
